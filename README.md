# Pokémon 3D Arena

個人版寶可夢卡片收藏 × 3D 對戰網站。拍實體卡入庫查價、寶可夢中心 3D 收藏館、歷代聯盟賽戰場即時對戰、點陣/動畫/現代 3D 三種畫風切換。

**僅供個人使用，不公開部署**（含任天堂版權素材）。
